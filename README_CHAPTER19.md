# Глава #19: Справка - Расширенная документация по игровой механике

## Введение
Данная глава представляет собой подробное руководство по игровым механикам без спойлеров к сюжету. Здесь описаны все расы, типы энергий, магические техники и система Soul UI.

---

## 1. Типы Энергий

### Человеческая энергия (Огонь)
- **Поведение**: Ведет себя как огонь
- **Применение**: Магия техники "вытачивается огнем"
- **Состояния**:
  - **Нормальное** (Огонь): Базовое состояние
  - **Искры**: При перепадах температур вылетают искры
  - **Человеческая молния**: Искры преобразуются в молнию (только сильные существа: Паракс, Парадоксер)
  - **Человеческий взрыв**: Теоретическое состояние, когда молнии скапливаются в огромной мощности (никто не использовал)
- **Усиление организма**: Можно использовать на ткани организма для усиления свойств

### Амальгамная энергия (Пластилин)
- **Поведение**: Ведет себя как пластилин
- **Особенности**: 
  - Обычно 2 магические техники
  - Возможность создания гибридов техник
  - Изменение структуры тела (физические атаки без магии бесполезны)

### Аномальная энергия (Вода)
- **Поведение**: Ведет себя как вода
- **Особенности**: Техника "растворена" в энергии, действие определяется во время использования
- **Ограничение**: Не могут сформировать заклинание в предметоподобную технику (коса, молот, шипы)

### Праническая энергия (Божественная)
- **Обладатели**: Божества, пробужденные существа хаоса
- **Особенность**: Безлимитное использование

### Энергия Хаоса
- **Обладатели**: Существ с поврежденной душой
- **Типы**: Ци, Проклятая энергия, Котики, Нитки, Вазилиновая энергия, Физическая выносливость

## 2. Расы

### Основные расы

#### 1. Человек
- **Энергия**: Человеческая
- **Тип души**: Нормальная
- **Особый прием**: Человеческая молния / Человеческий взрыв
- **Мастерство**: Некоторые сильные существа владеют молнией

#### 2. Амальгама
- **Энергия**: Амальгамная
- **Тип души**: Нормальная
- **Особый прием**: Гибридизация техник + изменение структуры тела
- **Количество техник**: 2 (можно создавать гибриды)

#### 3. Аномалия
- **Энергия**: Аномальная
- **Тип души**: Нормальная
- **Особый прием**: Владение Аномальной зоной
- **Типы АЗ**: Пустая, Простая, Техническая, Абсолютная

#### 4. Марионетка
- **Энергия**: Зависит от кукловода (поступает через нити)
- **Тип души**: Отсутствует
- **Правила**:
  - Энергия поступает через нити
  - Срыв с нитей = смерть
  - Урон по душе идет кукловоду (исключение: Джек имеет собственную душу)

### Обратные расы (Хаос)

#### 5. Заключенный человек
- **Энергия**: Хаос (Амальгамная + Аномальная)
- **Тип души**: Поврежденная
- **Особый прием**: [Магическая перезапись]
  - Объединение Аномальной и Амальгамной энергий
  - Изменение реальности под правило магии при касании
  - Пример: Техника Пыли → [Распыление] → существо рассыпается

#### 6. Раздельная амальгама
- **Энергия**: Хаос (Человек + Аномалия)
- **Тип души**: Поврежденная
- **Состав**: 5-10 существ, но только двое имеют энергию
- **После пробуждения**: Рассоединяются, выживают только Человек и Аномалия
- **Особый прием**: [Королевская зона]
  - Человек распространяет волю (Человеческую энергию)
  - Аномалия открывает Аномальную зону
  - Эффект: Подавление всех энергий + Человеческая молния по площади

#### 7. Сформированная аномалия
- **Энергия**: Хаос (Человеческая + Амальгамная)
- **Тип души**: Поврежденная
- **Ограничение**: Не способны отделяться в свою реальность
- **Особый прием**: [Кузница техники]
  - Создание любого оружия, предмета, артефакта, брони в рамках своей техники

### Прочие расы Хаоса

#### 8. Существо хаоса искаженной энергии
- **Энергия**: Уникальная (Ци, Проклятая, Котики, Нитки и т.д.)
- **Тип души**: Полностью поврежденная
- **Особенность**: Индивидуальная энергия со своими правилами

#### 9. Существо хаоса лишенное энергии
- **Энергия**: Отсутствует (до пробуждения)
- **Тип души**: Только оболочка
- **После пробуждения**: Безлимитная техника на основе Пранической энергии
- **Функционал**: Полный контроль (форма, свойства, правила в зоне)

### Существ высшего порядка

#### 10. Дети Богов Первого порядка (Системные люди)
- **Количество**: Ровно 6 (больше быть не может)
- **Энергия**: Праническая + одна другая
- **Особенность**: Содержат деталь от Божественного артефакта "Устройство"
- **Созданы**: От одного Божества

#### 11. Дети Богов Второго порядка
- **Энергия**: Праническая
- **Особенность**: Способны на Созидание и Разрушение в прямом смысле
- **Созданы**: От Богов напрямую

#### 12. Божество
- **Энергия**: Праническая (безлимитная)
- **Особенность**: Неубиваемые
- **Доступ**: [Консоль] и [Перезапись]

## 3. Аномальные Зоны (АЗ)

### Типы зон

#### Пустая АЗ
- **Затраты энергии**: Низкие (~10 ед.)
- **Мастерство**: Не требуется особого мастерства
- **Действие**: Нейтрализует техники внутри зоны
- **Эффективность**:
  - Плохо работает против техник Человека (имеют форму)
  - Размывает Амальгамную и Аномальную энергию
- **При столкновении с другой АЗ**:
  - Либо держится, блокируя чужую АЗ
  - Либо полностью подавляет чужую АЗ

#### Простая АЗ
- **Затраты энергии**: Средние (~20 ед.)
- **Мастерство**: Не требуется особого мастерства
- **Действие**: Пассивное применение магии в зоне
- **Визуал**: Пространство меняет цвет под магию

#### Техническая АЗ
- **Затраты энергии**: Высокие (~50 ед.)
- **Мастерство**: ~3 месяца тренировок
- **Действие**: Магия действует по определенным правилам или на определенные цели
- **Применение**: 
  - Механизмы и артефакты (Флаг подавления магии)
  - Личное использование (остановка времени только для магии)

#### Абсолютная АЗ
- **Затраты энергии**: Очень высокие (~200 ед.)
- **Мастерство**: Несколько десятков лет
- **Действие**: Полное искажение пространства (пассивное + техническое)
- **Известные названия**:
  - "Божественная чистка" (Коджекс)
  - "Танец лезвий" (Джек)
  - "Гниющий сад воспоминаний" (Росток)
  - "Бесконечная сплавка" (Барреро - не завершена)

## 4. Разделы магических техник

### Тема "Стихия"
Управление определенным ресурсом:
- Огонь, Вода, Лед, Космос, Кислота
- Природа, Песок, Воздух, Молния, Тьма

### Тема "Техническая"
Манипуляция технологиями, наука как магия:
- Хакер Реальности, Хирургия
- Манипуляция роботами, Лазер

### Тема "Пространство"
Сфера влияния - пространство:
- Гравитация, Телекинез, Телепорт

### Тема "Время"
Сфера влияния - время:
- Магия возврата, Хронос, Пропуск, Ускорение

**Примечание**: Магия Времени относится к теме "Время", даже если охватывает аспект мира

### Тема "Особая"
Выделяющиеся техники:
- Исцеление, Вампиризм, Плавление энергии
- Взрывы, Разрезы

### Тема "Божественная"
Охватывают аспекты мира:
- Созидание, Разрушение, Контроль, Разум, Отрицание

**Примечание**: Любое дополнение к аспекту магии (созидание материи, разрушение стекла, опустошение разума) относится к теме "Божественная"

## 5. Soul UI - Интерфейс Души

### Базовые кнопки (есть у всех)

| Кнопка | Цвет | Описание |
|--------|------|----------|
| **[Действие]** | Белый | Любое взаимодействие с миром. Пока существо живет, кнопка невредима |
| **[Атака]** | Красный | Физическая атака (кулак, меч). Может усиливаться магией |
| **[Защита]** | Синий | Блокировка опасности. Защищать можно не только себя |
| **[Пощада]** | Зеленый | Отказ от насилия, путь переговоров |
| **[Магия]** | Фиолетовый | Использование магии. Есть только у владеющих мастерством |
| **[Читы]** | Искаженный | Нелегальная слабая копия [Перезапись] |

### Божественные кнопки (ТОЛЬКО у Богов)

| Кнопка | Цвет | Описание |
|--------|------|----------|
| **[Консоль]** | Голубой | Отображение данных о мире/объекте. Только просмотр, без редактирования |
| **[Перезапись]** | Оранжевый | Глобальное редактирование Вселенной. Любая команда. Работает только в своей Вселенной |

**Примечание**: Системная пара #8 может использовать [Перезапись] в границах Вселенной #8, но [Консоль] работает везде

## 6. Система генерации персонажей

### Создание случайного персонажа
1. **Выбор расы** (по вероятностям):
   - Обычные (70%): Человек, Амальгама, Аномалия
   - Редкие (20%): Марионетка, Расы хаоса
   - Легендарные (9.9%): Дети Богов
   - Мифические (0.1%): Божество

2. **Настройка энергии** в зависимости от расы

3. **Выбор случайных техник** (1-3 техники):
   - Учитывается доступность тем для расы
   - Амальгамы получают 2 техники для гибридизации
   - Высшие расы имеют доступ к божественной магии

4. **Проверка пробуждения**:
   - Некоторые расы способны к пробуждению
   - Пробуждение дает новые возможности

5. **Настройка Soul UI**:
   - Добавляется кнопка [Магия] если есть способности
   - Божества получают [Консоль] и [Перезапись]

### Создание заклинаний
Заклинания создаются исходя из типа энергии:
- **Человеческая**: "Огненная X техника" (вытачивается огнем)
- **Амальгамная**: "Гибридная X техника" (смешивание техник)
- **Аномальная**: "Растворенная X техника" (техника в энергии)
- **Праническая**: "Божественная X техника" (прямое творение)
- **Хаос**: "Хаотическая X техника" (непредсказуемый эффект)

## 7. Установка и использование в Unity

### Структура проекта
```
Assets/
├── Scripts/
│   ├── EnergySystem.cs        // Система энергий
│   ├── MagicSystem.cs         // Магические техники
│   ├── RaceSystem.cs          // Система рас
│   ├── SoulUI.cs              // Интерфейс Soul UI
│   ├── AnomalyZone.cs         // Компонент Аномальной Зоны
│   ├── CharacterGenerator.cs  // Генератор персонажей
│   ├── PlayerController.cs    // Пример управления
│   └── PlayerStats.cs         // Характеристики игрока
```

### Настройка персонажа

1. **Создайте GameObject персонажа** в сцене

2. **Добавьте компоненты**:
   - `EnergySystem`
   - `MagicSystem`
   - `RaceSystem`
   - `SoulUI`
   - `CharacterGenerator` (опционально, для генерации)
   - `PlayerController` (для управления)
   - `PlayerStats` (для здоровья и характеристик)

3. **Настройте параметры** в Inspector:
   - Выберите начальную расу
   - Настройте максимальную энергию
   - Добавьте известные техники

4. **Для случайной генерации**:
   - Вызовите `CharacterGenerator.GenerateRandomCharacter()`
   - Или нажмите **G** в игре (если используется PlayerController)
   - Нажмите **R** для перегенерации

### Управление (PlayerController)

| Клавиша | Действие | Кнопка Soul UI |
|---------|----------|----------------|
| **ЛКМ** | Атака | [Атака] |
| **ПКМ** | Защита | [Защита] |
| **E** | Магия | [Магия] |
| **Q** | Особый прием расы | - |
| **F** | Действие | [Действие] |
| **T** | Пощада | [Пощада] |
| **G** | Генерация персонажа | - |
| **R** | Перегенерация | - |

### Пример кода

```csharp
// Использование кнопки Soul UI
soulUI.PressButton(SoulUIButtonType.Attack, targetVector, damage);

// Использование магии
magicSystem.UseTechnique(MagicType.Fire, targetPosition);

// Открытие Аномальной зоны (затраты энергии, не длительность!)
magicSystem.OpenZone(AnomalyZoneType.Simple, centerPosition, radius);

// Особый прием расы
raceSystem.UseSpecialAbility(parameters);

// Пробуждение
raceSystem.Awaken();
```

## 8. Советы по балансировке

### Затраты энергии для АЗ
- Пустая: 10 ед.
- Простая: 20 ед.
- Техническая: 50 ед.
- Абсолютная: 200 ед.

### Время изучения
- Пустая/Простая АЗ: Мгновенно
- Техническая АЗ: ~3 месяца тренировок
- Абсолютная АЗ: Несколько десятков лет

### Вероятности рас
Настройте в `CharacterGenerator`:
- `commonRaceChance`: 70f
- `rareRaceChance`: 20f
- `legendaryRaceChance`: 9.9f
- `mythicRaceChance`: 0.1f

### Количество техник
Настройте в `CharacterGenerator`:
- `minTechniquesCount`: 1
- `maxTechniquesCount`: 3

## 9. Важные примечания

### Аномальные Зоны
- **Не имеют длительности** - активны пока не будут деактивированы
- **Энергия тратится при создании**, не каждый кадр
- Зоны могут сталкиваться друг с другом

### Пробуждение
- Доступно некоторым расам (Хаос без энергии, Раздельная амальгама, и др.)
- После пробуждения открываются новые способности
- Раздельная амальгама после пробуждения разделяется на компоненты

### Марионетки
- Не имеют собственной души
- Энергия поступает через нити от кукловода
- Срыв с нитей = мгновенная смерть
- Урон по душе марионетки передается кукловоду

## Заключение

Эта система предоставляет гибкую основу для создания разнообразных персонажей с уникальными способностями. Каждая раса имеет свои особенности, ограничения и особые приемы, что создает глубокий стратегический слой в игре.

**Важно**: Данная документация не содержит спойлеров к сюжету и описывает только игровые механики.

Для начала работы:
1. Скопируйте все скрипты в папку `Assets/Scripts/`
2. Создайте пустой GameObject в сцене
3. Добавьте необходимые компоненты
4. Настройте параметры в Inspector
5. Запустите игру и нажмите **G** для генерации персонажа
